# 5G QoS Analyzer

![Language](https://img.shields.io/badge/Language-C++17-blue.svg)
![Platform](https://img.shields.io/badge/Platform-Windows%20%2F%20Linux-lightgrey.svg)

Uma aplicação CLI desenvolvida em **C++17** para simulação de tráfego de redes 5G e cálculo de métricas de Qualidade de Serviço (QoS) orientadas aos três pilares da recomendação ITU-R M.2083: **eMBB**, **URLLC** e **mMTC**.
